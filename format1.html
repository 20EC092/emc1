Web Application

A web application is a software program that runs on a web server and is accessed by users through a web browser using the internet. Web Development has two divisions: front-end development for the interface and user experience, and back-end development, which involves developing for the server, database, and application logic.

  1. Builds websites from scratch or using frameworks/tools
  2. Writes code using languages like HTML, CSS, JavaScript, and often backend languages like Python, PHP, Java, Node.js, etc.
  3. Works with databases, APIs, and servers
  4. Ensures the website works properly on different devices and browsers
